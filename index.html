Preguntas de Autoevaluación

¿Por qué usamos clamp() en los tamaños de fuente en lugar de valores fijos?

Porque clamp() permite fluidez entre un valor mínimo, preferido y máximo, adaptándose a diferentes anchos de pantalla sin perder control.

¿Qué pasaría si removemos el meta viewport de nuestro HTML?

Sin el meta viewport, los navegadores móviles escalan la página como si fuera de escritorio, causando zoom no deseado y mala legibilidad.

¿Por qué elegimos CSS Grid para la sección de skills en lugar de Flexbox?

CSS Grid facilita crear una cuadrícula bidimensional (filas y columnas), ideal para mostrar tarjetas organizadas, mientras que Flexbox es unidimensional.

¿Cuál es la diferencia práctica entre usar rem y em para espaciados?

rem se basa en el tamaño de fuente raíz (), garantizando consistencia global; em es relativo al tamaño de fuente del padre, útil para escalado local.
